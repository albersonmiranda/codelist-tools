# codelist-validator-rs

This is a rust package for validating codelists (SNOMED, ICD10, OPCS)

## Aims

`codelists-validator-rs` is a lightweight library for validating medical coding
standards including SNOMED CT, ICD-10, and OPCS-4.

## Features

- Validate medical codes against standard formats:
  - SNOMED CT codes
  - ICD-10 codes (including support for X-codes)
  - OPCS-4 codes
- Comprehensive error handling for invalid codes and data formats
